## npm 安装
推荐使用 npm 的方式安装，它能更好地和 [webpack](https://webpack.js.org/) 打包工具配合使用。

```shell
npm install i-ui -S
```

如果是通过 npm 安装，并希望配合 webpack 使用，请阅读下一节：<router-link to="/quickstart">快速上手</router-link>。
